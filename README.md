## Description

This project invites users to navigate a platformer game as an explorer. Players must collect keys to unlock a treasure chest while avoiding enemies - scorpions and poisonous plants.

---

## Interaction Instructions

- Press “Start Exploring” to begin the game.
- Use WASD or arrow keys to move.
- Press SPACE to attack scorpions.
- Getting hit by a scorpion or poisonous plant removes one life.
- Collect all 10 keys and reach the treasure chest at the end of the level with at least one life remaining to unlock it.
